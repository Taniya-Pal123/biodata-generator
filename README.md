# Biodata Generator (MERN Stack)

 Biodata Generator web application where users can register, login, and create their biodata easily.

## 🚀 Live Demo

Frontend:https://biodata-generator-frontend.onrender.com
Backend API:https://biodata-generator-backend.onrender.com

## 📂 Project Structure

biodata-generator
│
├── frontend (React)
└── backend (Node.js + Express)

## 🛠 Tech Stack

* Frontend: React, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* Authentication: JWT
* Deployment:  Render (Backend)

## ✨ Features

* User Registration
* User Login Authentication
* Secure Password Handling
* Create Biodata
* View Biodata
* Download Biodata



## 🔐 Environment Variables

Create a `.env` file inside the backend folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
